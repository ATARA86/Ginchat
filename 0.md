## 1. utils 包一般干什么用？
utils （工具包）是 Go 项目中存放 通用工具函数 的地方，类似于"工具箱"。

### 常见的 utils 包内容：
功能 说明 数据库初始化 连接 MySQL、Redis 等 配置文件读取 使用 Viper 读取 YAML/JSON 配置 加密/解密 MD5、SHA256、AES 等 时间处理 时间格式化、时区转换 字符串处理 截断、替换、验证 HTTP 工具 发送请求、JSON 解析 日志工具 统一日志输出格式


请求 → Redis缓存 → 命中 → 直接返回（不查数据库）
              ↓ 未命中
           查数据库 → 写入Redis → 返回

请求 → Redis INCR → 超限 → 拒绝（不查数据库）
              ↓ 未超限
           放行 → 查数据库

分布式锁
请求 → Redis INCR → 超限 → 拒绝（不查数据库）
              ↓ 未超限
           放行 → 查数据库

写请求 → Redis队列 → 异步 → 批量写入数据库


方式 保护数据库的原理 
缓存 减少数据库查询次数 
限流 拒绝非法请求，减轻压力 
分布式锁 保证操作串行化，避免并发问题 
消息队列 削峰填谷，批量写入


线程：创建成本 ~1MB内存，切换成本高
协程：创建成本 ~2KB内存，切换成本极低

1个 Go 程序可以轻松创建 数十万个协程 ，而线程可能只能创建几千个。
