## 什么时候使用路由组？
### 1. 当多个路由有共同前缀时
比如用户相关的接口：

- /user/getUserList - 获取用户列表
- /user/CreateUser - 创建用户
- /user/deleteUser - 删除用户
- /user/updateUser - 更新用户

当多个路由需要 相同的中间件 时


┌─────────────────────────────────────────────────────────┐
│                    用户登录流程                          │
└─────────────────────────────────────────────────────────┘

1. 用户提交: name=xxx, password=123
                     ↓
2. 服务器验证用户名密码
                     ↓
3. 生成 JWT Token (包含用户身份)
   Header: {"alg": "HS256"}
   Payload: {"identity": "tom", "exp": "2026-03-28 19:00:00"}
   Signature: HMACSHA256(Header + Payload, "secret key")
                     ↓
4. 返回给客户端: {"token": "eyJ..."}
                     ↓
5. 客户端保存 Token
                     ↓
6. 后续请求带上 Token
   Headers: Authorization: Bearer eyJ...
                     ↓
7. 服务器验证 Token 签名
   - 有效 → 允许访问
   - 无效/过期 → 拒绝访问
